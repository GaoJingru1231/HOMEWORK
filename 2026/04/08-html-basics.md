## Key Concepts

### Concept 1: HTML 是什么
**Explanation**: HTML（HyperText Markup Language）是制作网页的标记语言，不是编程语言。它负责网页的"结构"，就像乐高积木搭建房子一样。

**Analogy**: HTML 就像写信的格式：
- `<html>` = 信封
- `<head>` = 给邮递员看的信息
- `<body>` = 给朋友的正文内容

### Concept 2: HTML 基本结构
**Explanation**: HTML 使用"标签"来标记内容。标签用尖括号 `<>` 包裹，成对出现：
- 开始标签：`<p>`
- 结束标签：`</p>`（带斜杠 `/`）

**Code Example**:
```html
<h1>我的标题</h1>
<p>我的段落文字</p>
```

### Concept 3: 常用标签
**Explanation**:
- `<h1>` - 最大的标题（Heading 1）
- `<p>` - 段落（Paragraph）
- 文件保存扩展名为 `.html`
