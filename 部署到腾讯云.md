# 格式快调 - 腾讯云 Webify 部署指南

## 方案：腾讯云 Webify（云托管）

> Webify 是腾讯云的全托管 Web 服务，支持 Python 后端 + 静态前端，国内访问快。

---

## 第一步：推代码到 GitHub

```bash
# 在项目目录执行
cd C:\Users\25114\Desktop\HOMEWORK

# 提交所有改动
git add .
git commit -m "feat: 准备部署到腾讯云"
git push
```

> 如果 git push 失败（网络问题），可以：
> 1. 把仓库改成 HTTPS 协议：`git remote set-url origin https://github.com/GaoJingru1231/HOMEWORK.git`
> 2. 或者手动在 GitHub 网页上传文件

---

## 第二步：腾讯云 Webify 部署

1. 打开 [腾讯云 Webify 控制台](https://console.cloud.tencent.com/webify)
2. 点击「新建应用」
3. 授权绑定 GitHub，选择 `GaoJingru1231/HOMEWORK` 仓库
4. 配置：
   - **构建命令**：`pip install -r requirements.txt`
   - **输出目录**：`.`（根目录）
   - **启动命令**：`uvicorn server:app --host 0.0.0.0 --port $PORT`
   - **环境变量**：无需配置
5. 点击「部署」，等待 2-3 分钟

部署成功后，会得到一个 `*.webify.cloud` 的访问地址，所有人都可以访问。

---

## 注意事项

- `python-docx` 需要处理 `.docx` 文件，Webify 的临时目录是 `/tmp`
- 免费配额：每月 5GB 流量，足够小项目使用
- 如果流量超标，可以升级到付费版（按量计费）

---

## 备选方案：如果 Webify 不支持 Python

可以用 **腾讯云函数（SCF）** + **API 网关**：

1. 在 [腾讯云函数控制台](https://console.cloud.tencent.com/scf) 新建函数
2. 运行时选 `Python 3.11`
3. 上传 `server.py` + `requirements.txt`
4. 触发器选「API 网关」，开启「响应集成」
5. 前端 `index.html` 里的 API 地址改成云函数地址
